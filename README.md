## Installation Program Blog Konten
1. Buka dan start web server **Apache** dan database **MySQL**
2. **Ekstrak** proyek ke dalam web server xampp/laragon
3. Import database mysql di folder *database/smk_blog_db.sql*
4. Jalankan google chrome dan akses URL: "http://localhost/blog-konten/"


## Operation Program
1. Klik menu **Admin** pada navbar halaman blog konten
2. **Login** menggunakan akun demo
3. Masuk ke halaman Dasboard Admin
  - Dashboard => halaman utama
  - Tambah Post => tambah post / artikel baru
  - Semua Post => kelola post
  - Kategori => kelola kategori
  - Cleanup Orphaned => Cleanup Orphaned Images
  - Pengaturan => Pengaturan & Struktur Proyek

4. Jika klik menu admin pada dropdown navbar icon akun admin maka muncul menu profile
  - Profile => profile admin
